The Project... Debut

This project is part of the final year project at the University of Mauritius. It is compulsory to complete this project in order for students to graduate. Therefore, i had to either choose a project title or to come up with a proposed title. I approached a lecturer with a proposed title and we had a meeting and fast foward to the end of the meeting, we concluded on 'ERP Multiplayer Simulation Game'.

This project debut was on the 6th of September 2021 and is an on-going project. The 1st testing of hypothesis of the project is expected to be carried out at beginning of the second semester.

How it has been done?

The project was initially a mobile application whereby students would be ble to play the game with ease, irrespective on location. But why not have it on other platforms too? Therefore it is much better to be a web application as it can be used either on computers and also on mobile devices, thus giving the users a choice.

The website is fully responsive and is lightweight so as to make it easier on the hardware used to load it quickly. The application shall also be lightweight and responsive.
